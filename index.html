⚙️ Panel Settings
🤖 Provider API (Auto-Order Automation)

Yahan apne provider SMM panel ka API URL aur Key daalo. Customer ka order automatically provider pe place ho jayega.

📖 Setup kaise karo:
1. Provider panel pe login karo → API section mein jao → API Key copy karo.
2. Provider panel URL aur API Key yahan paste karo → Save karo.
3. Services add karte waqt Provider Service ID bhi daalo.
4. Ab customer order karega → automatically provider pe place ho jayega! ✅
📱 Telegram Notifications

Automatic background notifications to your Telegram bot — works alongside WhatsApp.

📖 How to set up:
1. Open Telegram & chat with @BotFather → send /newbot → follow steps → copy the Bot Token.
2. Get your Chat ID: chat with @userinfobot or @getidsbot and copy the ID.
3. For a group/channel: add your bot, then forward any message from there to @getidsbot to get its ID (starts with -100).
4. Paste both values above, enable the toggle, hit Save, then click 🧪 Send Test Message to verify.
